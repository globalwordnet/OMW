Load Wordnets from the command line

This page provides documentation for the script add-wn.sh (which calls validate-wn.py). These allow someone to add wordnets directly to the database (not through the web interface)

Usage

  
$ python3 scripts/validate-wn.py -h
  
usage: validate-wn.py [-h] [--auto-upload] [--no-upload] [--no-add-proj]
                      wnfile

A program to make wordnet LMF from OMW 1.0 tsv

positional arguments:
  wnfile         WN-LMF.xml(.gz) to be loaded

optional arguments:
  -h, --help     show this help message and exit
  --auto-upload  Automatically upload the WN if it validates (default: False)
  --no-upload    Don't ask to upload the WN even if it validates (default:
                 False)
  --no-add-proj  Don't automatically add the project (default: False)

add-wn.sh takes the same arguments and makes sure the environment is set up correctly

This script is used to load the automatically made wikitionary-wordnet files, the orphan wordnets, geowordnets and more.
