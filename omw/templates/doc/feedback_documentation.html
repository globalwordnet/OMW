Feedback about Individual wordnets

Open Multilingual Wordnet is a collection of different wordnets from around the globe that were independently created, some of them with limited resources and time frames, resulting in different accuracies.

To give your valuable feedback to the individual wordnets:

  • Visit the home page, there is a list of wordnets. In the feedback column there is the list of active email ids for feedback.
  • Use those email IDs to report an issue or to give feedback.
  • To get more information about individual wordnets, you can check the wordnet column of the same list to go to the project homepage (if it has one).

Feedback about the website

This website is maintained by the OMW community. The source code is hosted on Github.

If you want to suggest an enhancement, report a bug, have a maintenance issue, or want to request a new feature to be added, please raise an issue for that on Github.

You can also contribute to the Open Multilingual Wordnet project, this guide may help you get started.

Feedback/Questions about the OMW project as a whole

You can reach us through mail.

You can ask questions in the Github issue section and find the answers to many commonly asked questions in the frequently asked question section.

Frequently Asked Questions
Which parts of speech are used by the Open Multilingual Wordnet?

Open Multilingual Wordnet contains "open-class words": nouns, verbs, adjectives, and adverbs.

Where do you get wordnets for Open Multilingual Wordnet?

Wordnets in the OMW are independent open source projects, there is a good list of them on the Global Wordnet Association's Wordnets in the World page.
