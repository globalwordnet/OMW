OMW Search Interface

This page provides documentation for the web interface to the Open Multilingual Wordnet.

Search Function

At the top of each page in OMW, there is a search function. Type in any lemma into the search, press Enter. The results will be in the two languages selected to the right of the search bar.

The search function understands GLOB patterns, which means:

  • * matches anything (including nothing ). *dog* matches all lemmas with dog anywhere in it, e.g. dog, hotdog, dogmatic, endogenous, etc.,
  • ? matches any one character. ca? matches all lemmas that start with ca and contain one more character, e.g. cat, cab, can, etc
  • [xyz] matches a single character that is either x, y, or z. The characters inside of the [] can be any characters. [Ss] matches either s or S. This is how you do case insensitive searches for case-sensitive languages like English.
