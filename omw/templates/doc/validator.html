Global WordNet Converter and Validator

To link your wordnet to the Open Multilingual Wordnet, you should first convert your wordnet to the public Wordnet LMF document type definition (LMF DTD).

Global Wordnet Association provides a validator to validate whether your wordnet is in the desired format or not, and a converter to convert wordnets which are in the right format to different representations.

  • Converter

Converter

It converts your wordnet file format from Wordnet JSON, Wordnet DB, Open Multilingual Wordnet CSV, plWordNet, DebVisDic and W3C RDF to Wordnet LMF format.

To convert:

  • Go to the Global Wordnet converter and validator page. At the top left of the page choose the convert option and select the desired format for the input file (the one which you want to convert).

    • Select the desired output wordnet format (Wordnet LMF).




  • You can either copy-paste the content of your wordnet file in the â€œInput Wordnet Dataâ€ box, or you can upload a wordnet file by selecting the â€œBrowse..â€ button (We suggest to prefer browsing the file).

  • Before converting the wordnet, be absolutely sure this is the wordnet you wish to upload! If so, press Convert.

    • If there are no errors in the passed file, wordnet in the desired format will be returned.
