Global WordNet Converter

To link your wordnet to the Open Multilingual Wordnet, you should first convert your wordnet to the public Wordnet LMF document type definition (LMF DTD).

Global Wordnet Association provides a converter to convert wordnets which are in the right format to different representations, and a validator to validate whether your wordnet is in the desired format or not.

  • Converter
  • Validator

Converter

It converts your wordnet file format from Wordnet JSON, Wordnet DB, Open Multilingual Wordnet CSV, plWordNet, DebVisDic and W3C RDF to Wordnet LMF format.

To convert:

  • Go to the Global Wordnet converter and validator page. At the top left of the page choose the convert option and select the desired format for the input file (the one which you want to convert).

    • Select the desired output wordnet format (Wordnet LMF).




  • You can either copy-paste the content of your wordnet file in the “Input Wordnet Data” box, or you can upload a wordnet file by selecting the “Browse..” button (We suggest to prefer browsing the file).

  • Before converting the wordnet, be absolutely sure this is the wordnet you wish to upload! If so, press Convert.

    • If there are no errors in the passed file, wordnet in the desired format will be returned.



    🎉🎉🎉 Congratulations! 🎉🎉🎉 You have successfully converted your wordnet in the desired format. 😊

Validator

It is an external validator. You will need to validate your wordnet file before uploading it to the Open Multilingual Wordnet by the local validator. It's not necessary to get your file validated here to upload it to the OMW.

To validate the format of your wordnet:

  • Visit the Global Wordnet converter and validator page, at the top left of the page choose the validate option and select the desired format.

  • You can copy-paste the content of your wordnet file in the “Wordnet Data” box, or you can upload a wordnet file by selecting the “Browse..” button. (We recommend you to prefer browsing the file, especially when the file is large).

  • Before validating the wordnet, be absolutely sure this is the wordnet you wish to upload! If so, press Validate.

  • If there are no errors in the passed file, “Validation OK!” will be flashed



  • If unfortunately an error flashes, then you need to debug your file, the message returned might help you figuring out the error.



  • Hopefully, you will be successful in debugging the file. After debugging you again need to press Validate, “Validation OK!” should be flashed this time.

  • 🎉🎉🎉 Congratulations! 🎉🎉🎉 You have successfully validated a wordnet 😊
Notes
  • Your wordnet file might get validated here, but it might not get validated by the local validator (e.g. in case your wordnet resource is forming cycles in the hypernym graph after getting merged). To be able to upload your wordnet resource, you need the approval of the local validator. This external tool is for your reference only.
  • You might get a “Too large request or file” message if you are copy-pasting a large wordnet file in the “Wordnet Data” box.



    • If you may browse the file instead of using the box facility, it might actually get validated.

  • If your wordnet file is big, it might take longer than the usual to load the result. It might be possible that the “Bad Gateway” message will get flashed, despite your wordnet being in the correct format.



    • In that case, you may again try to validate your wordnet (validator isn’t perfect, as of now you should be successful in validating the wordnet in few attempts).
